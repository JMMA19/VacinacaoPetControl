# VacinacaoPetControl

precisa:
    node versão 20.15.0
    npm versão  10.7.0
    nvm versão 1.1.12

Como Rodar:
   1 clone o repositorio
   2 Configure seu .env
   3 configure seu banco de acordo com seu .env
   4 restaure o banco.
   5 execute o "npm install --force"
   6 execute "npm start:debug"
